Network request failed: Este error lo tuve en mi celular usando expo go, desde la web las peticiones se hacian
al servidor pero ahi no. La solucion fue cambiar "localhost" en las peticiones por la direccion IP de mi peticiones
Luego instale "npx expo install expo-build-properties" y en app.json agregue el plugin para que se puedan enviar
solicitudes HTTP. En produccion LO DEBO SACAR SI O SI
